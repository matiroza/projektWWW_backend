# projektWWW_backend
#### Projekt został napisany w środowisku MERN (MongoDB, Express JS, React JS i Node JS).
###### Aplikacja typu Strava, która pozwala na logowanie się za pomocą konta Google.
###### Użytkownik ma możliwośc dodawania aktywności prywatnej lub publicznej ( http://localhost:3000/activities/add ).
###### Podgląd swoich aktywności oraz opcje edycji oraz opcje ich usuwania ( http://localhost:3000/dashboard ).
###### Użytkownik może przejść do sekcji public activities ( http://localhost:3000/activities ) gdzie może zobaczyć wszystkie publiczne aktywności użytkowników.
